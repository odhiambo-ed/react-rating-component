# React Star Rating Component

## Description

> This project demonstrates how to build a star rating component in React. The component allows users to rate items by clicking on stars, with hover effects to preview their rating. The first star is not active by default, ensuring a clear user experience.

## Features

- Dynamic star rating
- Hover effects to preview rating
- Click to set rating
- First star is not active by default

### Run

